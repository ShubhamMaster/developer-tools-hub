# Security Policy

## Supported Versions

The `main` branch is the only supported version.

## Reporting a Vulnerability

Please report security issues via GitHub Security Advisories:
https://github.com/ShubhamMaster/developer-tools-hub/security/advisories/new

If you are unable to use advisories, open a private message to the maintainer
with details and steps to reproduce.

We will confirm receipt within 72 hours and provide a timeline for remediation
as soon as possible.
